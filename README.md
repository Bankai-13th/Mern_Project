**NOTE MANAGEMENT SYSTEM**

A full-stack Note Management System web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js) with Bootstrap for responsive styling. This project allows users to create, manage, and organize their notes efficiently with a clean and user-friendly interface.

**Features**
- Dashboard – Centralized view of all notes with easy navigation.
- Create Notes – Add new notes with dynamic form validation.
- Profile Page – Manage user details and view personalized information.
- Authentication – Secure login and logout functionality with token-based authentication.
- Responsive Design – Styled with Bootstrap for mobile-friendly layouts.

- **Tech Stack**
-  Frontend: React.js, Bootstrap, HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Architecture: MVC pattern with secure client-server communication

**Project Structure**
- Dashboard – Displays all user notes.
- CreateNotes – Form to add new notes.
- Profile – User profile management.
- Logout – Securely ends user session.

**Purpose**
This project demonstrates a practical implementation of the MERN stack with authentication, CRUD operations, and responsive UI design. It is a great starting point for building scalable note-taking or task management applications.
